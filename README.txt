This is a Spring-boot microservice, with a DDD architecture based in N-layers, which starts an API to get Prices

Instructions:

    in eclipse, add the project as maven project, make a maven clean package, and run with spring boot "run" the PortalApplicattion.java class.
    in cmd, java -jar Portal-0.0.1-SNAPSHOT (which is in the target directory).

    You also can try SWAGGER-UI in the following route : http://localhost:8080/swagger-ui.html

java version -> 1.8
